Cada nivel, representa un game.stage y se manejan mediante game.state. Un nivel, puede ser una variable cualquiera! pero game espera que esta variable tenga los tres métodos

En la línea 28, 29 de ./{{version.directory}}/game.js, es donde le indicamos a game, que level1 es el primer nivel del juego.

Cada uno de los métodos de level1, tienen su función:

preload

En el preload, se cargan todos los recursos que vas a usar en el juego, y se ejecuta al principio de todo, una sola vez.

create

El create se ejecuta una sola vez, luego de que todos los recursos del preload hayan sido cargados. Es ideal para crear al personaje principal,

update
