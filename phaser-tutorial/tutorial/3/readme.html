Creando un sprite

El sprite es el bloque básico de construcción, en phaser. Un sprite puede tener física, una imagen, y animaciones.

El sprite es una buena representación de un personaje, en el juego. Así que el sprite tiene el método kill(), que hace que el sprite desaparezca, desactiva las colisiones, y otras propiedades, y también tiene el método reset(posicionX, posicionY), para resetear al personaje en una posición, volviendo a activar todas sus propiedades y colisiones
